# Dokyll

[![CI Status](https://github.com/dokyll/dokyll/actions/workflows/ci.yml/badge.svg)](https://github.com/dokyll/dokyll/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Dokyll è un generatore di siti statici leggero e veloce che trasforma i tuoi progetti in pagine web pronte da pubblicare.

È pensato per essere semplice da usare, facilmente estendibile e adatto a qualsiasi tipo di progetto basato su file.

---

## Funzionalità

- Generazione di siti statici da cartelle progetto
- Supporto a più progetti
- Build veloce e leggera
- Utilizzo da linea di comando
- Struttura semplice e personalizzabile

---

## Installazione

Scarica l’ultima versione di Dokyll dal sito ufficiale:

https://dokyll.github.io/

Dopo aver scaricato il programma, apri il terminale e verifica l’installazione:

```bash
dokyll
```

Se tutto funziona correttamente, vedrai il menu principale di Dokyll.

---

## Utilizzo

Per generare un sito statico:

```bash
dokyll build-pages mio-sito
```

Questo comando crea la build del progetto e genera il sito pronto per essere pubblicato.

---

##  Output

Dokyll genera un file di build contenente il sito pronto all’uso, mantenendo la struttura del progetto originale.

---

##  Come funziona

Dokyll legge la cartella del progetto, elabora tutti i file e crea una build finale che può essere utilizzata per la pubblicazione o l’integrazione con altri sistemi.

---

##  Perché Dokyll

- Leggero e veloce
